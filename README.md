#  MUSA - Galería de Arte Digital

##   Integrante

* Limachi Quispe Manuel Alejandro

---

## Descripción

MUSA es una aplicación web desarrollada para la administración de artistas, obras de arte y usuarios mediante una arquitectura Frontend-Backend.

Proyecto desarrollado para la materia **Web III**.

---

## Tecnologías Utilizadas

### Frontend

* React
* Bootstrap

### Backend

* NodeJS
* MySQL

### Seguridad

* JWT
* bcrypt

---

## Funcionalidades

* CRUD de Artistas
* CRUD de Obras
* Eliminación lógica de registros
* Login con JWT
* CAPTCHA
* Contraseñas encriptadas
* Registro de logs de acceso
* Reporte PDF
* Gráficos estadísticos
* Gestión de usuarios

---

##  Seguridad Implementada

* Autenticación mediante JWT
* CAPTCHA para acceso al sistema
* Contraseñas cifradas con bcrypt
* Control de acceso por roles
* Registro de ingreso y salida de usuarios

---

##  Reportes y Estadísticas

* Generación de reportes PDF de artistas
* Visualización de gráficos estadísticos
